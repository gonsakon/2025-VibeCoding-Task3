關於我的開發旅程

我原本是非本科背景，因為很想做出真正能被使用者看見、實際解決問題的產品，開始自學前端開發。

自學的過程中，我透過線上課程與實作多個小型專案，一步步累積 HTML、CSS、JavaScript 能力，也在這段歷程裡確認自己想正式往前端工程師方向轉職。

目前主要使用 HTML、CSS、JavaScript 開發網頁，持續練習 RWD 版型與前端效能優化，希望能加入實際的商業團隊，一邊參與專案，一邊把程式碼品質拉到更專業的水準。

  • 自學半年以上，完成多個 RWD 網站作品
  • 習慣把設計稿拆解成有結構的版型與元件
  • 熟悉 Git / GitHub 版控與基本協作流程
  • 樂於主動溝通，會先把需求寫成清楚規格再開始寫程式
